详细教程

深入了解如何配置和使用各项功能

Windows 设备配置

在 Windows 设备上配置 DNS 设置，享受广告拦截服务。

  1. 1
    打开网络设置
    右键点击任务栏的网络图标，选择"打开网络和Internet设置"
  2. 2
    进入适配器选项
    点击"更改适配器选项"，找到您当前使用的网络连接
  3. 3
    配置DNS服务器
    右键网络连接 → 属性 → Internet协议版本4(TCP/IPv4) → 属性
  4. 4
    输入DNS地址
    选择"使用下面的DNS服务器地址"，输入我们提供的DNS服务器地址
# 推荐的DNS服务器地址
首选DNS服务器: 8.8.8.8
备用DNS服务器: 8.8.4.4

移动设备配置

在 iOS 和 Android 设备上配置 DNS 设置。

  1. 1
    iOS 设备配置
    设置 → Wi-Fi → 点击已连接网络的 (i) 图标 → 配置DNS → 手动
  2. 2
    Android 设备配置
    设置 → 网络和互联网 → Wi-Fi → 长按网络名称 → 修改网络 → 高级选项
  3. 3
    输入DNS地址
    在DNS设置中输入我们提供的服务器地址，保存设置

路由器配置

在路由器上配置 DNS，为整个网络提供保护。

  1. 1
    登录路由器管理界面
    在浏览器中输入路由器IP地址（通常是192.168.1.1或192.168.0.1）
  2. 2
    找到DNS设置
    在网络设置或WAN设置中找到DNS服务器配置选项
  3. 3
    配置DNS服务器
    输入我们的DNS服务器地址，保存并重启路由器

使用管理面板

了解如何使用我们的管理面板查看统计信息和配置设置。

  1. 1
    登录管理面板
    使用您的账户登录到我们的管理面板
  2. 2
    查看统计信息
    在仪表板中查看DNS查询统计、拦截记录等信息
  3. 3
    自定义设置
    根据需要配置过滤规则、白名单等个性化设置

常见问题

解答您在使用过程中可能遇到的问题

配置DNS后网速会变慢吗？
不会。我们的DNS服务器采用全球分布式架构，响应速度通常比ISP提供的DNS更快。 而且通过拦截广告和恶意内容，实际上会提升您的浏览体验。
如何知道DNS配置是否生效？
配置完成后，您可以访问一些包含广告的网站，如果广告被成功拦截， 说明配置已生效。您也可以在我们的管理面板中查看实时统计数据。
某些网站无法访问怎么办？
如果某些网站被误拦截，您可以在管理面板的白名单中添加该域名。 我们也提供临时禁用过滤的选项，方便您快速解决访问问题。
支持哪些设备和系统？
我们的服务支持所有能够配置DNS的设备，包括Windows、macOS、Linux、 iOS、Android、路由器等。无需安装任何软件。
如何恢复原来的DNS设置？
在DNS设置中选择"自动获取DNS服务器地址"或输入您ISP提供的DNS地址即可恢复。 具体操作与配置过程相反。
服务是否免费？
我们提供免费的基础服务，包括广告拦截和基本统计。 高级功能如详细分析、自定义规则等需要升级到付费计划。
